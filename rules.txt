Projeto de Gerenciamento de Estoque em Python:

Sets: Utilize um conjunto (set) para armazenar os nomes dos produtos, garantindo a unicidade dos itens no estoque.

Dicionários: Use um dicionário para representar o estoque, onde as chaves são os nomes dos produtos e os valores são outros dicionários contendo informações como preço e quantidade em estoque.

Generators: Utilize um generator para iterar sobre os itens do estoque.

Decorators: Implemente decorators, como um para contar o número de chamadas de métodos e outro para lidar com erros.

Tratamento de Erros: Adicione tratamento de erros para lidar com situações como a remoção de um produto que não existe no estoque.

itertools: Importe a biblioteca itertools para utilizar funcionalidades como o contador (count) para contar o número de chamadas de métodos.
